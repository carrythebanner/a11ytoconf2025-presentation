What we don't (yet)

We need to be more holistic with our concept of responsiveness. There's still a tendency to assume there's only "one true presentation" in terms of:

  • Font selection and size
  • Colors
  • Media features like animation preferences
  • "Cross-platform" patterns that don't match OS/platform norms
  • Keyboard/gesture shortcuts, common interaction patterns (e.g. Back button placement and appearance)
  • Arbitrary user customizations, e.g. user stylesheets, ad blockers, or other extensions
